# Portafolio de Desarrollo Web - Parciales

Bienvenido a mi repositorio de prácticas y parciales de Desarrollo Web. Este proyecto es una demostración integral de mis habilidades de front-end, combinando estructuración semántica, diseño moderno de interfaces (UI), e interactividad mediante JavaScript.

## 🚀 Tecnologías y Características Principales

El proyecto ha sido recientemente rediseñado para adoptar un enfoque **Premium**, utilizando un estilo visual moderno conocido como **Glassmorphism**, combinando un elegante tema oscuro con gradientes radiales, sombras traslúcidas, y tipografías limpias.

### 🎨 Tecnologías Utilizadas:

- **HTML5**: Estructuración semántica robusta para la web.
- **CSS3**:
  - Variables CSS (`Custom Properties`) para temas globales.
  - Uso intensivo de **Flexbox** y **CSS Grid** para la adaptación a múltiples resoluciones (Responsive Design).
  - Filtros visuales (`backdrop-filter: blur`) para generar el estilo "Glassmorphism" (vidrio esmerilado).
  - Animaciones y transiciones sutiles (Glow y Hover effects).
- **JavaScript (Vanilla JS)**:
  - Manipulación del DOM en tiempo real.
  - Algoritmia lógica para validaciones de seguridad mediante Expresiones Regulares (`RegExp`).
  - Cálculos y generación de estructuras interactivas.
- **Librerías / Dependencias**:
  - **SweetAlert2** (Notificaciones modales e interactivas).
  - **FontAwesome** (Íconos vectoriales).
  - **Google Fonts** (`Outfit` e `Inter`).

## 📂 Estructura del Proyecto

El sitio principal (`index.html`) actúa como la página de inicio al portafolio. Cuenta con una barra de navegación (Navbar) interactiva estilo _Sticky_, y está estructurado en módulos para demostrar:

- **Estructuras HTML5 y JS Fundamentos**: Secciones interactivas "Hero" con recursos.
- **Flexbox y Grid**: Una amplia zona de documentación visual demostrando propiedades de alineación para Flexbox con elementos dinámicos.
- **Parciales / Proyectos Específicos**: Colección de problemas comunes de JS y CSS resueltos con interfaces de usuario pulidas de alta fidelidad.

### 📝 Resumen de Parciales

A continuación, una breve lista de los parciales alojados en el directorio `html/`, demostrando diferentes resoluciones de problemas:

1. **Parcial 1 (Calculadora Numérica)**: Sistema asíncrono estilizado en una tarjeta para calcular operaciones de resta secuenciales.
2. **Parcial 2 (Generador de Tablas de Multiplicar)**: Un generador de listas dinámicas con validación de "multiplicaciones cero" y captura de errores sin recargar la página.
3. **Parcial 4 (Validación de Seguridad)**: Formulario estilo credencial que requiere, por medio de Expresiones Regulares en JavaScript, que el usuario jamás introduzca números en campos exclusivos de texto alfabético.
4. **Parcial 5 (Formulario de Registro Complejo)**: Formulario tipo _Grid_ de 2 columnas que valida completitud, concordancia de contraseñas dobles y renderiza una "ID Card" estilizada con el resumen de los datos validados.
5. **Parcial 7 (Enlaces 3D)**: Botones interactivos que se expanden a un diseño isométrico en 3 dimensiones con sombras _glow_ al interactuar.
6. **Parcial 8 (Blog Posts Grid)**: Grilla (CSS Grid) responsiva demostrando tarjetas de publicaciones interactivas modernas con jerarquía tipográfica premium.

## 🛠 Instalación y Despliegue Local

Para experimentar y previsualizar completamente las animaciones e importar correctamente las bibliotecas de JavaScript:

1. Clona el repositorio:

   ```bash
   git clone https://github.com/JordyPradaYanes/Parciales_DesWeb.git
   ```

2. Ejecuta un servidor local en el mismo directorio. Se recomienda encarecidamente la herramienta `live-server` por vía de NPM o mediante extensión de VS Code.
   ```bash
   npx live-server
   ```
   _(Esto levantará el proyecto en `http://127.0.0.1:8080` de manera fluida y con hot-reload incluido)._
