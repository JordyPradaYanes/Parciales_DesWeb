🔌 Demo Fetch API

Busca cualquier Pokémon por nombre o número y observa cómo JavaScript consume una API externa en tiempo real.

❌ Pokémon no encontrado. Intenta con otro nombre o número.

#001

Bulbasaur

💡 Concepto Demostrado: fetch() + async/await

Al hacer clic en "Buscar", se llama a fetch('https://pokeapi.co/api/v2/pokemon/' + nombre). La respuesta se convierte en JSON con .json(), y los datos se usan para actualizar dinámicamente el HTML. También se maneja el error con try/catch para mostrar un mensaje si el Pokémon no existe.
