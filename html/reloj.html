💡 Concepto Demostrado: setInterval()

Este reloj usa setInterval(updateClock, 1000) para ejecutar la función updateClock cada 1000ms (1 segundo). Dentro de esta función, se obtiene la hora actual con new Date() y se actualiza el contenido del elemento HTML con textContent. ¡Así funciona la actualización dinámica del DOM!
